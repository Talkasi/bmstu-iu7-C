- `program st SRC_FILE DST_FILE`
- `program ft FILE SUBSTR`
- `program at FILE`
## Positive tests
- Sort products without equal price;
- Sort products with equal price;

## Negative tests
- No arguments;
- Wrong parameters;
- Input file is empty;
- Wrong reading-file name;
- All product names do not end with a substring.
